# Using TradingView Strategy Auto Trading Bot (使用TradingView上的策略來進行自動化交易)
這是一個透過Discord、TradingView、幣安來達成的自動交易機器人。
<h1>目的</h1>
在網路上看到許多影片，但是大部分的影片都很複雜，於是就自己動手做一個自動交易機器人。
<h1>使用到的技術</h1>
Python</br>
API</br>
Discord</br>

*Azure VM*</br>
*Linux*
